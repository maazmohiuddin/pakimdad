Get Started

1

Basic uploader

Bulkit features the awesome Innostudio File uploader, a very flexible premium uploader component built with jQuery. You can build the uploader you've always dreamed of ! Many options let you customize the look and feel of your component. Check out the documentation and the above examples to get and idea fo what you can do.

2

Field uploader

You can display the file uploader as an upload input field. Every selected item is displayed in a list below the uploader. You can remove selected items individually.

3

Thumbnails uploader

The example above shows how to manage multiple uploads at once using a thumbnail list. Every file can also be removed individually.

4

Drag' n Drop uploader

A visually interesting upload form that also handles drag and drop functionnality. Selected items are shown in a list below the uploader.
